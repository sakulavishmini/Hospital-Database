# Suwapiyasa Hospital Database Mini Project

## Overview
This is a **Hospital Database Management System** for the private hospital **“Suwapiyasa”** in Colombo, developed as a university mini project.  
It manages staff, patients, medications, surgeries, and hospital locations using **MySQL**. The project includes **ER diagrams, relational schema, triggers, views, stored procedures, functions, and XML data import**.

---

## Features
- Manage **staff**: Doctors, Surgeons, Nurses (with specialties, contracts, salaries).  
- Manage **patients**: Personal info, location, allergies, primary care doctor.  
- Track **medications**: Quantities, costs, interactions, expiry.  
- Manage **surgeries**: Surgeons, nurses, theatre, date/time, special needs.  
- **Views** for patient-surgery info.  
- **Triggers** to auto-update medication info.  
- **Stored procedures & functions** for medication tracking and expiry checks.  
- Import staff and patient data from **XML files**.
